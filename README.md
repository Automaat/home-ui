# @mskalski/home-ui

Shared UI components for home ecosystem apps.

## Features

- Svelte 5 components
- Nord theme + Open Props
- TypeScript support

## Installation

```bash
npm install @mskalski/home-ui
```

## Usage

```typescript
import { Card, CardHeader, CardTitle, CardContent, Modal } from '@mskalski/home-ui';
```

Import styles:

```css
@import '@mskalski/home-ui/styles';
```
